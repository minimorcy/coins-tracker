# Crypto Price Tracker

Este proyecto es un rastreador de precios de criptomonedas construido con Angular y el tema de administración de CoreUI. Proporciona una interfaz limpia y moderna para visualizar datos de criptomonedas en tiempo real.

## Características

*   Visualización de precios de criptomonedas.
*   Gráficos históricos de precios (usando Chart.js).
*   Panel de control moderno y receptivo.
*   Basado en Angular 20 y CoreUI v5.

## Empezando

Sigue estas instrucciones para obtener una copia del proyecto en funcionamiento en tu máquina local para desarrollo y pruebas.

### Prerrequisitos

Necesitarás tener Node.js y npm instalados en tu sistema. Se recomienda usar las versiones especificadas en `package.json`:

```sh
node: ^20.19.0 || ^22.12.0 || ^24.0.0
npm: >= 9
```

### Instalación

1.  Clona el repositorio:
    ```sh
    git clone https://github.com/tu-usuario/coins-tracker.git
    ```
2.  Navega al directorio del proyecto:
    ```sh
    cd coins-tracker
    ```
3.  Instala las dependencias de npm:
    ```sh
    npm install
    ```

### Ejecutando la Aplicación

Para iniciar el servidor de desarrollo de Angular, ejecuta el siguiente comando. La aplicación se abrirá automáticamente en tu navegador en `http://localhost:4200/`.

```sh
npm start
```

## Scripts Disponibles

En el directorio del proyecto, puedes ejecutar:

*   `npm start`: Inicia la aplicación en modo de desarrollo.
*   `npm run build`: Compila la aplicación para producción en la carpeta `dist/`.
*   `npm test`: Ejecuta las pruebas unitarias con Karma.
